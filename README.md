## Run Locally

Clone the project

```bash
  git clone https://github.com/NajehRouin/Home-Fitness.git
```

Go to the project directory

```bash
  cd Home-Fitness
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
